### خطوات التثبيت
## انسخهم بالتسلسل
1.  `apt-get update && apt-get upgrade -y`
2.  `apt-get install wget -y`
3.  `apt-get install proot -y`
4.  `apt-get install git -y`
5.  `cd ~`
6.  `git clone https://github.com/sajjad-salam/ubuntu.git`
7.  `cd ubuntu`
8.  `chmod +x ubuntu.sh`
9.  `./ubuntu.sh -y`
10. `./startubuntu.sh`
